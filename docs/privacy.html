VIT Mobile プライバシーポリシー

最終更新日: 2026年5月3日

VIT Mobile（以下「本アプリ」）は、ユーザーの音声入力をテキストに変換し、他のアプリに挿入するためのツールです。本ポリシーは、本アプリがどのようなデータを収集・使用するかを説明します。

1. 収集する情報

本アプリは、以下の情報をユーザーが本アプリを使用する間にのみ取得します。

  • 音声データ: マイクボタンを押している間に録音される音声。録音は文字起こし API に送信された後、端末上でもサーバー上でも永続化されません。
  • 文字起こし結果テキスト: 音声から変換されたテキスト。クリップボードへのコピーおよび端末内の履歴ファイル（history.json）への保存のために使用します。
  • API キー: ユーザーが入力した Groq API キー（および任意で Anthropic API キー）。端末内の SharedPreferences に保存され、外部には送信されません（ただし、各 API への通信時には HTTPS で当該サービスに送信されます）。

2. 第三者サービスへの送信

本アプリは、文字起こしと任意の文章補正のため、以下の第三者サービスに音声およびテキストを送信します。

  • Groq Inc. (プライバシーポリシー): 音声データを whisper-large-v3-turbo モデルに送信し、テキストに変換します。
  • Anthropic, PBC (プライバシーポリシー): 文章補正機能（オプション）を有効にした場合のみ、文字起こし結果テキストを Claude モデルに送信します。デフォルトでは無効です。

本アプリは独自のサーバーを持ちません。ユーザーのデータは本アプリの開発者には送信されません。

3. 端末内に保存される情報

  • API キー: SharedPreferences
  • 履歴: アプリ内部ストレージの files/history.json
  • 設定: SharedPreferences

これらは本アプリのアンインストール時にすべて削除されます。

4. 権限

  • マイク: 音声入力のため。
  • 他のアプリの上に表示: フロートマイクアイコンの表示のため。
  • ユーザー補助サービス: 文字起こし結果を他のアプリのフォーカスされたテキスト入力欄に挿入するため。本アプリは入力欄の内容を読み取りません。ACTION_PASTE イベントの送信のみを行います。
  • インターネット: 第三者 API への通信のため。

5. 子供のプライバシー

本アプリは 13 歳未満の子供を対象としていません。

6. ポリシーの変更

本ポリシーは予告なく変更されることがあります。重要な変更がある場合は、アプリ内またはこのページで通知します。
