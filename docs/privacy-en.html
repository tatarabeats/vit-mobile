VIT Mobile Privacy Policy

Last updated: May 3, 2026

VIT Mobile ("the App") is a tool that converts your voice input into text and inserts it into other apps. This policy describes what data the App collects and uses.

1. Information Collected

The App accesses the following information only while you are actively using it:

  • Audio data: Audio recorded while the microphone button is pressed. Recordings are sent to a transcription API and not persisted on the device or on any server after transcription.
  • Transcribed text: Text converted from your voice. Used for clipboard copy and saved to the device's local history file (history.json).
  • API keys: The Groq API key (and optionally Anthropic API key) you enter. Stored locally in SharedPreferences and not transmitted to any party except the corresponding API service over HTTPS.

2. Transmission to Third-Party Services

The App transmits audio and text to the following third-party services for transcription and optional text correction:

  • Groq Inc. (Privacy Policy): Audio data is sent to the whisper-large-v3-turbo model for transcription.
  • Anthropic, PBC (Privacy Policy): Only if you enable the optional text correction feature, transcribed text is sent to the Claude model. Disabled by default.

The App does not operate its own servers. Your data is not transmitted to the App developer.

3. Information Stored on Device

  • API keys: SharedPreferences
  • History: files/history.json in the app's internal storage
  • Settings: SharedPreferences

All of the above are deleted when you uninstall the App.

4. Permissions

  • Microphone: For voice input.
  • Display over other apps: To show the floating microphone icon.
  • Accessibility Service: To insert transcribed text into the focused text field of other apps. The App does not read the content of input fields. It only sends ACTION_PASTE events.
  • Internet: For communication with third-party APIs.

5. Children's Privacy

The App is not intended for children under 13.

6. Changes to This Policy

This policy may be updated without prior notice. Material changes will be notified in the App or on this page.
